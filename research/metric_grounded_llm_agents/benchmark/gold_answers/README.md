# Gold Answers

Gold answer JSON files are generated by:

```bash
python3 research/metric_grounded_llm_agents/benchmark/generate_gold_answers.py
```

This directory is intentionally empty in source control because gold answers
must be regenerated from the real local CMS PY2026 DuckDB/dbt marts. In the
current Codex runtime, the required public CMS/HealthData CSV downloads returned
HTTP 403 and no local DuckDB database was present, so gold answers were not
generated.
