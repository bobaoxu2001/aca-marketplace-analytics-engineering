# Gold Answers

Gold answer JSON files are generated by:

```bash
python3 research/metric_grounded_llm_agents/benchmark/generate_gold_answers.py
```

Generated gold-answer JSON is intentionally ignored by source control and must
be recreated from the validated local CMS PY2026 DuckDB/dbt marts. Runtime agent
modules do not read this directory; evaluation code loads the gold rows only
after a system has produced its output. Dated compact summaries and provenance
records are versioned under `artifacts/` without committing the generated local
gold-answer directory.
