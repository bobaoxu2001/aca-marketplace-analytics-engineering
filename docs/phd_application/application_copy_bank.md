## Approximately 150-word research summary

Natural-language analytics agents can produce executable SQL while silently
using the wrong measure, grain, grouping, population, or output contract. In an
independent project, I built a reproducible DuckDB/dbt warehouse over six
official CMS Marketplace files containing 3.89 million source rows, designed a
30-question benchmark, and implemented an evaluation pipeline separating metric
routing, query compilation, policy validation, execution, strict result
agreement, and evidence reporting. After discovering that
projection-compatible scoring overstated exact correctness, I introduced a
strict complete-result contract and rescored the saved runs. In an
oracle-table-conditioned subscription-Codex pilot, all 90 SQL queries executed,
but only 3 reproduced the complete reference result. A benchmark-specific
compiler matched 24 of 30 source questions, then fell to 12 of 30 on a locked,
model-generated lexical-shift set even with oracle routing. These diagnostic
results motivate my doctoral agenda: independently validated cross-schema
benchmarks, learned metric and schema grounding with calibrated abstention, and
human studies of evidence-first analytics interfaces.

## Statement-of-purpose project paragraph

My interest in trustworthy AI grew from a practical failure mode: an analytics
agent can return syntactically valid SQL and a fluent answer while silently
using the wrong metric or grain. To study this problem, I developed
Metric-Grounded Analytics Agents, a reproducible benchmark over official CMS
Marketplace data that separates metric routing, constrained compilation,
execution, strict result comparison, and evidence reporting. While auditing the
evaluation, I found that an alias-tolerant metric was counting some incomplete
outputs as exact. I replaced it with a complete schema-and-row contract and
rescored all applicable stored SQL/result outputs. Although 90/90 generated SQL
queries executed, only 3/90 matched the complete reference result. That
experience shifted my focus from aggregate accuracy to validity-aware failure
analysis: how agent failures can be measured, localized, and communicated
responsibly. In doctoral work, I want to study independently validated
cross-schema evaluation, learned semantic grounding with calibrated abstention,
and human interaction with evidence-first data agents.
